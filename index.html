How It Works

  1. Send a Request

    Make a GET request to our `/convert` endpoint with your image URL and optional width and height parameters.

  2. Image Processing

    Our server fetches the image, converts it to AVIF format, and resizes it if dimensions are provided.

  3. Receive the Image

    You receive the converted image in AVIF format, ready for use in your applications or websites.

  4. Caching for Speed

    Nginx caches frequently requested images, reducing processing time and improving response speed.

API Endpoint

Use our straightforward API to integrate image conversion into your projects.

GET /convert?url=<IMAGE_URL>&width=<WIDTH>&height=<HEIGHT>

Public API Endpoint

GET https://imageconvertapi.adgstudios.co.za/convert?url=<IMAGE_URL>&width=<WIDTH>&height=<HEIGHT>

Parameters:


  • url (required): The URL of the image to convert.
  • width (optional): Desired width of the image.
  • height (optional): Desired height of the image.

The response will be the converted image in AVIF format.
